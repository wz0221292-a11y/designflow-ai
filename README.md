# DesignFlow AI 使用教程

## 📌 访问地址

**线上版**：https://designflow-ai-chi.vercel.app

**本地开发**：http://localhost:3000

---

## 🚀 快速开始

### 1. 注册账号

1. 打开网站，点击「注册」
2. 填写姓名、邮箱、密码（至少6位）
3. 点击注册
4. 如果邮箱验证已开启，会提示你前往邮箱验证；如果已关闭，则直接进入系统

> 💡 如果出现"email rate limit"提示，说明注册过于频繁，等几分钟后再试

### 2. 登录

1. 输入邮箱和密码
2. 点击登录，进入项目仪表板

---

## 📁 项目管理

### 创建项目

1. 在仪表板页面，点击右上角「新建项目」
2. 在弹窗中描述你的产品想法（如："一款智能宠物喂食器"）
3. 点击「创建」，系统自动进入项目编辑页

### 编辑项目

1. 在项目卡片上点击「编辑」按钮
2. 可以修改项目名称和产品描述
3. 点击「保存」

### 删除项目

1. 在项目卡片上点击「删除」按钮
2. 确认删除（不可恢复）

## 🎨 7步设计流程

进入项目后，按顺序完成 7 个设计步骤，每步都有 AI 自动生成内容：

### 步骤 1：背景研究
AI 自动分析市场痛点、技术趋势、竞品机会。你可以手动编辑修改。

### 步骤 2：产品介绍
AI 生成产品名称、定位语、核心功能、使用场景。

### 步骤 3：用户画像
AI 生成 2-3 个典型用户画像，包含年龄、职业、需求、痛点、使用场景。
点击「+ 添加画像」可手动添加更多。

### 步骤 4：外观设计
AI 生成产品效果图（3张）。点击图片可预览大图，点击「重新生成」可单独重新生成某张。

### 步骤 5：CMF方案
AI 推荐色彩、材料、表面处理方案，包含主色和辅色的色值。

### 步骤 6：故事板
AI 生成产品使用场景故事板（6格），每格可添加场景描述。

### 步骤 7：爆炸图
AI 生成产品结构分解图。

---

## 🎮 操作按钮说明

每个步骤底部有 3 个按钮：

| 按钮 | 功能 |
|------|------|
| **重新生成** | 强制重新生成当前步骤内容（覆盖现有内容） |
| **跳过此部分** | 跳过当前步骤，不生成内容，导出时也不包含 |
| **满意，下一步 →** | 保存当前内容并进入下一步（自动生成下一步内容） |

> 💡 第一次进入新步骤时，AI 会自动生成内容，无需手动点击。
> 💡 跳过的步骤不会出现在最终导出中。
> 💡 你可以随时点击顶部进度条回到已完成的步骤。

---

## 📤 导出

完成所有步骤后，点击「导出」进入导出页面：

- **PPT 演示文稿**：导出 .pptx 文件，包含所有步骤内容
- **PDF 展板**：导出 .pdf 文件，适合打印和展示

> 💡 只有已完成的步骤才会出现在导出中，跳过的步骤不会包含。

## 🔧 常见问题

### Q: 注册后登录提示"Email not confirmed"？
在 Supabase 后台 → Authentication → Settings 中关闭「Enable email confirmations」。

### Q: 输入框文字看不清？
所有输入框已使用深色字体，如果仍有问题请刷新页面。

### Q: 图片生成失败？
图片生成使用外部 API，需要配置 `IMAGE_API_KEY` 和 `IMAGE_API_ENDPOINT`。请检查 `.env.local` 文件中是否正确配置了 API 密钥和端点。

### Q: 导出 PDF 打不开？
已修复，现在使用 jsPDF 生成标准 PDF 格式。

## 🛠 本地开发与部署

### 本地运行

```bash
cd designflow-ai
npm install
npm run dev
# 访问 http://localhost:3000
```

### 重新部署到 Vercel

修改代码后，运行：

```bash
vercel deploy --prod
```

### 下线服务器

1. 打开 https://vercel.com/dashboard
2. 进入项目 → Settings → General
3. 滚动到底部，点击「Delete Project」

> 删除后可以随时重新 `vercel deploy --prod` 上线

## 📋 技术栈

- **前端**：Next.js 16 + React 19 + Tailwind CSS
- **后端**：Next.js API Routes
- **数据库**：Supabase PostgreSQL
- **AI文本**：DeepSeek API
- **AI图片**：外部图像生成 API（可配置端点）
- **PPT导出**：PptxGenJS
- **PDF导出**：jsPDF
- **部署**：Vercel（免费）
